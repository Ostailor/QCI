# Data

This directory stores generated synthetic microgrid patch datasets and any documented external data references added in later implementation phases.

The default reproduction command writes:

- `generated_case.yaml`: full synthetic case definition
- `microgrids.csv`: one-row-per-microgrid engineering summary
- `scenarios.csv`: scenario probabilities and stress metadata
- `benchmarks/pglib_case5_pjm_adapted.yaml`: optional public-benchmark-derived CMPO case generated by `python scripts/run_benchmark.py --seed 42 --quick`

The default data is synthetic research data only. It is not private utility data and is not suitable for operational grid planning.
